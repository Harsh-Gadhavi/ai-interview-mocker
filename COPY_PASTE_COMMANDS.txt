# DEPLOYMENT COMMANDS - Copy & Paste One By One

Open PowerShell and run these commands in order. **Copy the entire command, paste it, and press Enter.**

---

## COMMAND 1: Install Vercel CLI
```
npm install -g vercel
```
**Status:** Wait until you see "added X packages" ‚úì

---

## COMMAND 2: Navigate to Project  
```
cd "c:\Users\harsh\nextjs React\ai-interview-mocker"
```
**Status:** Prompt should show your project path ‚úì

---

## COMMAND 3: Login to Vercel
```
vercel login
```
**Status:** Browser opens, sign in, come back to terminal ‚úì

---

## COMMAND 4: Deploy to Production
```
vercel --prod
```

**When prompted in terminal, answer:**
- "Set up and deploy?" ‚Üí Type: `Y` and press Enter
- "Which scope?" ‚Üí Press Enter (or select your account)
- "Link to existing project?" ‚Üí Type: `N` and press Enter
- "What's your project's name?" ‚Üí Press Enter
- "In which directory is your code?" ‚Üí Press Enter
- "Want to override the settings below?" ‚Üí Type: `N` and press Enter

**Status:** Wait 3-5 minutes, you'll see the production URL ‚úì

---

## COMMAND 5: Get Your Deployment URL
Once deployment finishes, copy the URL shown (looks like: `https://ai-interview-mocker.vercel.app`)

**Status:** Save this URL, it's your live app! ‚úì

## ‚ö†Ô∏è CRITICAL: Add Environment Variables

**You MUST do this or the app won't work!**

1. Go to: https://vercel.com/dashboard
2. Find: `ai-interview-mocker` project
3. Click: **Settings**
4. Click: **Environment Variables**
5. Click: **Add new**
6. Name: `NEXT_PUBLIC_CLERK_PUBLISHABLE_KEY`
   Value: (Copy from https://clerk.com/dashboard ‚Üí Settings ‚Üí API Keys)
7. Click: **Save** and **Add new** again
8. Name: `CLERK_SECRET_KEY`
   Value: (Copy from https://clerk.com/dashboard ‚Üí Settings ‚Üí API Keys)
9. Click: **Save**
10. Click: **Redeploy** button at the top
11. Wait 2-3 minutes for redeploy

**Status:** Your app is now fully configured! ‚úì
